primeiro clonar a pasta do github com o comando   :
git clone https://github.com/FabioBD7/MarinaAdevaLTDA.git 
 entrar na pasta criada.
 cd MarinaAdevaLTDA

Apos criar ou editar algum arquivo ou pasta adicionar a alteração com o comando:
git add {nome do arquivo ou pasta }

 
 efetuar um comit do que foi feito com o comando:
	$ git commit -m "texto sobre o comit"
	
	###
	ao realizar o comit pela primeira vez recebi mensagens 
	solicitando que colocasse  e-mail e nome com os respectivos comandos:

git config --global user.email "you@example.com"
  git config --global user.name "Your Name"
  
	###
	
	

Subir as alterações com o comando:
$ git push
